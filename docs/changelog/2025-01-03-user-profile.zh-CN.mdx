# 用户数据统计与活跃度分享 💯

想要了解自己在 LobeChat 上的活跃度表现吗？

现在，您可以通过数据统计功能，全方位了解自己的 AI 数据，还可以生成个人活跃度分享图片，与好友分享您在 LobeChat 上的活跃度。

## 📊 数据统计

- **数据统计**: 助手数 / 话题数 / 消息数 / 累计字数
- **排行版**:
  - 模型使用率 `Top 10`
  - 助手使用率 `Top 10`
  - 话题内容量 `Top 10`
- **热力图**: 过去一年内的活跃度分布
- **用户活跃度分享**: 生成个人活跃度分享图片

## 👉 使用方式

1. 需要使用 `PgLite` 或 `数据库` 模式
2. 点击个人头像进入「账户管理」-「数据统计」页面
