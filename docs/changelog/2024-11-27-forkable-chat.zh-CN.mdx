# 重磅推出分支对话功能 🎉

我们很高兴地宣布，LobeChat 推出了全新的分支对话功能，让您的对话体验更加流畅自然：

## 核心特性

- **消息分支**: 支持在任意消息处创建新的对话分支
- **双模式切换**:
  - 延续模式 (Continuation): 保持原有上下文继续探讨
  - 独立模式 (Standalone): 基于选定消息开启全新话题

## 使用方式

1. 在任意消息右侧点击「创建分支」按钮
2. 开始新的对话分支
