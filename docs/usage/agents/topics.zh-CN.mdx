- **保存话题：** 在聊天过程中，如果想要保存当前上下文并开启新的话题，可以点击发送按钮旁边的保存按钮。
- **话题列表：** 点击列表中的话题可以快速切换历史对话记录，并继续对话。你还可以通过点击星标图标 <kbd>⭐️</kbd> 将话题收藏置顶，或者通过右侧更多按钮对话题进行重命名和删除操作。
